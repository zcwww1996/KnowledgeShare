# 3. 隐式转换
简单说，隐式转换就是：当Scala编译器进行类型匹配时，如果找不到合适的候选，那么隐式转化提供了另外一种途径来告诉编译器如何将当前的类型转换成预期类型。

隐式转换有四种常见的使用场景：
1. 将某一类型转换成预期类型
2. 类型增强与扩展
3. 模拟新的语法
4. 类型类
